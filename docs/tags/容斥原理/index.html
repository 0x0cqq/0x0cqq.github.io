容斥原理
「CF83D」Numbers-容斥原理
📅 Jan 9, 2019 · ✏️ 791 words · ☕ 2 mins read

给出三个整数 $l,r,k (1 \le l \leq r \le 2\cdot10^9, 2 \le k \le 2 \cdot 10^9)$ 。

求在区间 $[l,r]$ 内满足 $k \mid i$ ， 且对于任意 $j \in [2,k-1]$ 都不满足 $k \mid i$ 的数 $i$ 的个数。
